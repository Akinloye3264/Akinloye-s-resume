## Usage

- **Personalized Greeting:** Enter your name in the form on the home page to receive a greeting based on the current time.
- **Dark Mode:** Click the moon icon in the header to toggle dark mode.
- **View Projects:** Navigate to the Projects page to see highlighted works.
- **Contact:** Use the contact form to send a message or view contact details.
- **Resume:** Download or view the resume using the provided buttons.
- **Profile Image Modal:** Click the profile photo to view it in a larger modal.

## Customization

- **Update Content:** Edit the HTML files to change text, links, or add new sections.
